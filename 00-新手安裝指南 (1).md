# OpenCode 新手安裝指南：從零開始

> 版本：v0.1
> 更新日期：2026-06-10
> 適用對象：完全沒有接觸過 OpenCode 的同學

> 📌 **這份指南是你所有教材的起點。** 就算你從沒聽過 OpenCode、不知道什麼是 API Key、沒用過 PowerShell，都沒關係——照著步驟做，一步一步來。

---

## 這份指南會幫你裝什麼？

全部裝好之後，你的電腦會有這些工具：

| 工具 | 用途 | 什麼時候會用到 |
|------|------|--------------|
| **OpenCode 桌面版** | 你的 AI 程式設計助教 | 每次上課、寫程式、整理資料 |
| **API Key**（通行證） | 讓 OpenCode 能連接 AI 模型 | 裝好 OpenCode 的第一件事 |
| **Git** | 記錄檔案修改過程（像遊戲存檔） | 第 4–5 課 |
| **GitHub CLI (gh)** | 跟 GitHub 雲端溝通（像遙控器） | 第 5 課 |
| **uv** | 管理 Python 工具（像 App Store） | 第 6 課以後 |

---

## 課前確認

在開始之前，請確認你的電腦符合以下條件：

- [ ] 你的電腦是 **Windows 10 以上** 或 **macOS**
- [ ] 電腦有 **網路連線**（Wi-Fi 或網路線）
- [ ] 你有 **Google 帳號**（沒有也可以先裝，但第 6 課會用到）

> 如果你用的是學校電腦，可能需要管理員權限才能安裝軟體。遇到權限問題請跟老師說。

## Step 1：安裝 OpenCode 桌面版

### ① 下載安裝檔

1. 打開瀏覽器（Chrome、Edge 都可以）
2. 在網址列輸入：**https://opencode.ai**
3. 進入官網後，找到 **Download** 按鈕點下去
4. 選擇你的作業系統版本：
   - Windows → 下載 `.exe` 安裝檔
   - macOS → 下載 `.dmg` 安裝檔

### ② 安裝

1. 雙擊下載好的安裝檔
2. 一直按「下一步」（Next）直到安裝完成
3. 安裝完成後，桌面會出現 OpenCode 的圖示

### ③ 開啟 OpenCode

雙擊桌面圖示開啟 OpenCode。第一次開啟時，你會看到一個對話畫面。

**檢查點 □** OpenCode 桌面版安裝完成

---

## Step 2：取得 API Key（通行證）

OpenCode 需要一組「API Key」才能連接 AI 模型。我們用 **OpenRouter** 來取得，它就像一個 AI 模型的大賣場。

### ① 註冊 OpenRouter

1. 打開瀏覽器，前往：**https://openrouter.ai**
2. 點右上角的 **Sign Up**（註冊）
3. 可以用 Google 帳號快速註冊，或用 Email 註冊
4. 登入後，點右上角你的頭像 → **Keys**
5. 點 **Create Key**
6. 為你的 Key 取一個名字，例如：`我的OpenCode`
7. 按 **Create** 後，會出現一串很長的文字

> ⚠️ **這串文字就是你的 API Key，複製下來，等一下要用！**

> 🔐 **注意：API Key 就像你的密碼，不要傳給別人，不要截圖發到群組！**

### ② 將 API Key 貼入 OpenCode

1. 回到 OpenCode 桌面版
2. 在對話框中輸入 `/connect` 並送出
3. 選擇 **OpenRouter**
4. 貼上你剛剛複製的 API Key
5. 按 Enter 確認

**檢查點 □** API Key 設定完成

## Step 3：第一次對話

API Key 設定好之後，來跟 OpenCode 說第一句話吧！

### 練習 1：打招呼

在對話框中輸入：

```
Hello OpenCode！我是新手，請多多指教！
```

### 練習 2：問問簡單問題

```

```

### 練習 3：請它解釋一個概念

```

```

> 如果 OpenCode 回答了你的問題，恭喜你——OpenCode 安裝成功了！

**檢查點 □** 與 OpenCode 成功對話

## Step 4：安裝環境工具（Git、gh、uv）

接下來要安裝三個開發工具。這三個工具在後面的課程會用到，現在先裝起來。

### 打開 PowerShell

> 📌 **PowerShell 是什麼？** 它是 Windows 內建的「文字指令操作介面」。你不用理解它怎麼運作，照著打指令就好。

1. 按鍵盤上的 **Win + R**（Windows 鍵 + R 鍵）
2. 輸入 `powershell`，按 Enter
3. 會跳出一個藍色底的視窗——這就是 PowerShell

> 💡 **小提醒**：如果之後遇到「指令找不到」的錯誤，先關掉 PowerShell 再重新打開。

### ① 安裝 Git

在 PowerShell 中輸入以下指令，然後按 Enter：

```powershell
winget install --id Git.Git --accept-source-agreements --accept-package-agreements
```

會看到一些文字在跑，等待它完成（可能需要 1–3 分鐘）。

安裝完成後，確認 Git 可以正常使用：

```powershell
git --version
```

應該會看到類似這樣的畫面：
```
git version 2.4x.x.windows.x
```

**檢查點 □** Git 安裝完成

### ② 安裝 GitHub CLI（gh）

在 PowerShell 中輸入：

```powershell
winget install --id GitHub.cli --accept-source-agreements --accept-package-agreements
```

等待安裝完成後，確認版本：

```powershell
gh --version
```

應該會看到類似：
```
gh version x.x.x
```

**檢查點 □** GitHub CLI 安裝完成

### ③ 安裝 uv

在 PowerShell 中輸入：

```powershell
powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"
```

安裝完成後，**關閉 PowerShell 再重新打開**，然後確認版本：

```powershell
uv --version
```

應該會看到類似：
```
uv x.x.x
```

**檢查點 □** uv 安裝完成

## 卡關診斷室

安裝過程中可能會遇到以下狀況，別緊張，這裡有解法：

| 狀況 | 原因 | 解法 |
|------|------|------|
| 輸入指令後出現「無法辨識」 | 指令打錯了，或工具還沒安裝 | 檢查指令有沒有打錯字，尤其是空格和連字號 |
| API Key 貼上後 OpenCode 說連線失敗 | Key 可能無效或貼錯了 | 回到 OpenRouter 網站檢查 Key 的狀態，重新複製貼上 |
| winget 安裝時卡住很久 | 網路下載較慢 | 等待 2–3 分鐘，如果還是沒回應就關掉 PowerShell 重來 |
| 出現「拒絕存取」或「權限不足」 | 需要管理員權限 | 關閉 PowerShell，按右鍵 → 以系統管理員身分執行 |
| 安裝後指令仍然找不到 | 沒有重開 PowerShell | 先關掉 PowerShell，再重新打開 |
| OpenCode 安裝後無法開啟 | 安裝檔可能損壞 | 重新下載安裝檔，再安裝一次 |

### 還是卡住怎麼辦？

1. 把錯誤訊息**完整複製**下來
2. 對 OpenCode 說：「幫我看這個錯誤訊息是什麼意思——（貼上錯誤訊息）」
3. 或者問旁邊的同學、舉手問老師
